What Changed from Base

  • base memakai satu struktur logic besar; v11 memecah eksekusi menjadi InvictusBullM5_v11, InvictusBearM5_v11, dan InvictusCombinedM5_v11.
  • Timeframe execution resmi menjadi M5, bukan M15, agar entry lebih cepat dan trade count naik.
  • Order live dipisah arah: bull-only hanya BUY, bear-only hanya SELL. Combined tester dipakai untuk validasi portfolio dalam satu report.
  • Score gate dilonggarkan menjadi V10_MinTradeScore=68 agar M5 tidak terlalu ketat.
  • Zone retest dibuat lebih agresif: ZoneLookback=8, ZoneMinBodyRatio=0.46, ZoneBreakATR=0.02, ZoneMidTouchATR=0.10, ZoneOvershootATR=0.35.
  • Risk profile dinaikkan: RiskPercent=5.00, MaxPositions=6, dan WeakRegimeMaxPositions=2.
  • Cross-direction guard aktif: bull magic 2026042411 akan membaca bear magic 2026042412, dan sebaliknya.
  • Comment trade tetap audit-friendly: B11|engine|Sscore|grade|tags untuk bull, S11|... untuk bear, dan C11|... untuk combined tester.
  • Dynamic SL/TP tetap berbasis swing/zone + ATR buffer, lalu RR disesuaikan engine. Breakeven tetap aktif di 0.80R.
  • Partial/trailing TP manager masih tersedia sebagai input, tetapi default final false karena pada ronde ini profit leader masih lebih baik tanpa TP manager.
