# Use This File

Untuk MT5/VPS, file yang dipakai adalah:

`MQL5/Experts/Acane/AcaneM1_v2.ex5`

Cara pakai:

- Copy folder `MQL5` dari package ini ke folder data MetaTrader 5.
- Di chart `XAUUSD,M1`, attach EA `AcaneM1_v2`.
- Pastikan local MT5 tidak menjalankan EA yang sama bersamaan dengan VPS.
- Setelah attach, migrate/sync ke MQL5 VPS.
