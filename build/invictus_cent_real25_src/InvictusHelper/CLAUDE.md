# CLAUDE.md

## Project Overview

InvictusHelper is a MetaTrader 5 custom indicator that visually displays all signals generated by the Invictus EA strategy on the chart. It serves as a companion tool for analysis and backtesting.

**Purpose**: Show historical and live SMC (Smart Money Concepts) signals with Entry, SL, TP levels, and trade outcomes — persistent on chart.

## Architecture

```
InvictusHelper.mq5          — Main indicator (OnCalculate loop)
Modules/
  IH_Config.mqh             — Inputs, enums, structs (IH_Signal, IH_Zone, IH_BOSInfo)
  IH_Detector.mqh           — BOS detection, zone detection, FVG, sweep, retest
  IH_SignalEngine.mqh       — Confluence scoring, trade simulation, signal storage
  IH_Renderer.mqh           — All chart objects (lines, labels, rectangles, dashboard)
```

## Signal Detection Pipeline (per bar)

```
1. Session check (Asia/London/NY windows)
2. Bias detection (H4 HH/LL trend, default bullish for gold)
3. BOS detection (fractal swing + break classification)
4. S/D Zone detection (Candle A/B pattern)
5. Confluence scoring (Zone+40, Vol+15, BOS+25, Trend+20, Gold+10 = max 110)
6. Grade A >= 80 → Signal
7. Trade simulation (fill check → TP/SL walk-forward)
```

## Detection Logic (mirrors Invictus EA v1.2.x)

- **BOS**: Fractal swing (lookback 2), close beyond swing, min break ATR×0.35
- **Zone**: Candle B body engulfs Candle A body, search window 1-6 bars, max $100 range
- **SL**: Zone boundary + ATR buffer, clamped $30-$85
- **TP1**: Entry ± SL×1.5 (RR 1.5)
- **Fill**: Limit order at 50% body B, expiry 2 H4 bars

## Chart Objects

All objects use prefix `IH_` for cleanup. Key object types:
- `IH_S{id}_ZONE` — Zone rectangle (blue=demand, red=supply)
- `IH_S{id}_ENTRY` — Entry level line (white dashed)
- `IH_S{id}_SL` — Stop loss line (red dotted)
- `IH_S{id}_TP` — Take profit line (green dotted)
- `IH_S{id}_RES_LBL` — Result label (TP HIT/SL HIT/EXPIRED/ACTIVE)
- `IH_SESS_*` — Session backgrounds
- `IH_DASH_*` — Dashboard panel

## Key Design Decisions

- **H4 timeframe**: Designed for H4 (matches EA). Works on other TFs with warning.
- **Persistent objects**: All chart objects are permanent, not deleted between ticks.
- **Historical scan**: On first load, scans up to 1500 bars (configurable).
- **Trade simulation**: Walk-forward from signal to determine TP/SL outcome.
- **P&L display**: Shown as dollar distance (per 0.01 lot equivalent).
- **AsSeries convention**: All arrays use AsSeries=true (0=newest bar).
